# 更新记录

## 0.0.2

类型：个人测试版本，尚未正式上线的第二个更新。

### 新增

1. `XiceCommandControl` 新增 `/xcc reload`，允许在游戏内重新加载插件配置。
2. `XiceCommandControl` 新增 `/xcc list [玩家名或UUID]`，允许查询玩家拥有的受控指令。

### 调整

1. 玩家专属权限改为以 UUID 登记，玩家名仅用于显示和查询。
2. `ExamplePlayer` 获得 `/creative`、`/xcc reload`、`/xcc list` 权限。
3. 部署脚本会同步自制插件的默认配置到服务器运行目录。

## 0.0.1

类型：个人测试版本，尚未正式上线的第一个更新。

### 新增

1. 确立服务器第一阶段目标：先启动基础 Java 版原版生存服务器。
2. 世界范围暂定为 `main`、`main_nether`、`main_the_end`。
3. 暂不加入 RPG、资源世界、PVP 专用世界、副本和复杂经济系统。
4. 建立版本号文件 `VERSION`。
5. 建立 Git 提交规范文档。

### 说明

版本号格式暂定为 `个人测试版本.上线阶段.更新序号`。

当前 `0.0.1` 表示：

1. 第一个 `0`：个人测试版本。
2. 第二个 `0`：尚未正式上线。
3. 第三个 `1`：第一个更新。
